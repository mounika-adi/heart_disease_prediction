Project Title:
Heart Disease Prediction Using Machine Learning

Description:
This project implements a machine learning–based heart disease prediction system.
The proposed FS-Ensemble model combines feature selection with ensemble learning
to improve prediction accuracy.

Tools and Technologies:
 1.Python
 2.Pandas
 3.NumPy
 4.Scikit-learn
 5.Matplotlib

Dataset:
Heart Disease dataset obtained from a publicly available source.

How to Run the Code:
1. Install required libraries using pip.
2. Open the Python file in any IDE (PyCharm / VS Code / Jupyter).
3. Run the file to preprocess data, train models, and generate results.
4. Output visualizations will be displayed on the screen.

Files Included:
1.Main_Report.pdf : Research paper document
2.Case_Study.pdf : Case study document
3.heart_prediction.py : Source code file
4.README.txt : Project description and instructions
